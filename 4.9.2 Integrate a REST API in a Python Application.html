Step 1: MapQuest 개발자 계정 및 API 키 발급
  • MapQuest 개발자 사이트(developer.mapquest.com)에 로그인합니다. [1]
  • Key Management 페이지에서 새 키를 생성하고 별도로 복사해둡니다. [1]
⚠️ 주의: 발급받은 API Key는 외부에 노출되지 않도록 주의하며, 실습 스크립트의 key 변수에 할당하여 사용합니다. [1]

Step 2: 기본 Python 스크립트 설계

필요한 라이브러리를 임포트하고 기본 변수들을 정의합니다. [1]

03
Step 3: API 호출 및 데이터 파싱

응답받은 JSON 데이터를 분석하여 거리(Distance)와 소요 시간(Duration)을 출력합니다. [1]

Core LogicPython
1json_data = requests.get(url).json()
2json_status = json_data["info"]["statuscode"]
3
4if json_status == 0:
5    print("API Status: " + str(json_status) + " (Success)")

7    print("From: " + orig)

💡 계산 팁: MapQuest는 기본 단위로 마일(Miles)을 사용합니다. 한국식 표현을 위해 1.60934(약 1.61)를 곱하여 킬로미터(KM)로 변환해줍니다. [1]
