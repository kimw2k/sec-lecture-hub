03
Part 4: Python을 활용한 대량 데이터 생성 자동화

반복적인 API 호출을 스크립트로 자동화하여 도서 100권을 일괄 등록합니다. [1]

add_books.pypython
1import requests
2
3url = "http://library.demo.local/api/v1/books"
4
5for i in range(1, 101):
6    payload = {"title": f"Book {i}", "author": "AI Bot"}
7    response = requests.post(url, json=payload)
8    print(f"Added {i}: {response.status_code}")
본 문서는 Cisco DEVASC 실습 가이드를 기반으로 재구성되었습니다.
Tip: 인쇄 시 설정에서 '배경 그래픽 항목'을 체크해야 모든 컬러가 정상 출력됩니다.
