AI 협업: FastAPI 구축 및 공공 데이터 시각화

교과과정: K-Move School DevNet 1.0 - 백엔드 개발 및 데이터 분석
필수도구: Python 3.10+, FastAPI, Pandas, Matplotlib
01
환경 구성: 가상환경 및 의존성 설치

프로젝트 독립성을 위해 파이썬 가상환경을 생성하고 필요한 라이브러리를 설치합니다.

$ python -m venv venv
$ source venv/bin/activate # Windows: venv\Scripts\activate
$ pip install fastapi uvicorn requests pandas matplotlib
02
FastAPI 초기 설정 및 헬스 체크

가장 기본적인 API 서버를 구동하여 환경이 정상적인지 확인합니다.

from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "FastAPI 서버 정상 작동 중"}

서버 실행 명령:

$ uvicorn main:app --reload
03
Swagger UI를 통한 인터페이스 테스트

FastAPI가 자동 생성하는 대화형 문서(Swagger)를 활용하여 API를 검증합니다.

  • 브라우저에서 http://127.0.0.1:8000/docs 접속
  • GET / 항목 클릭 → [Try it out] 버튼 클릭
  • [Execute] 클릭 후 Response Body에서 결과값 확인
04
응용: 기상청 데이터 연동 및 시각화

한국 기상청 단기예보 API를 호출하고, 데이터를 정제하여 그래프로 표시합니다.

🤖 AI 프롬프트 추천 (데이터 요청 및 처리) "공공데이터포털 기상청_단기예보 조회서비스 API를 Python requests로 호출하는 코드를 작성해줘. 1. 결과(JSON)에서 'T1H(기온)' 항목만 추출해.
2. Pandas를 사용해 시간별 기온 데이터프레임을 만들어.
3. Matplotlib을 이용해 x축은 시간, y축은 기온인 선 그래프(Line Chart)를 그리는 함수를 작성해줘.
4. 한글 폰트 깨짐 방지 코드도 포함해줘."

필수 구현 요건:

# AI 작성 코드 검토 포인트
1. Service Key: API 오픈 키 보안 관리 여부
2. Base Date/Time: 현재 시점 기준 동적 생성 여부
3. Error Handling: API 응답 실패 시 예외 처리
