03
Swagger UI 인터페이스 테스트

FastAPI의 자동 생성 문서를 통해 API의 정상 응답 여부를 테스트합니다.

💡 테스트 절차:
  1. 주소창에 http://127.0.0.1:8000/docs 입력
  2. 목록에서 GET / 클릭 → Try it out 버튼 클릭
  3. Execute 실행 후 밑 하단 Response Body에서 JSON 데이터 확인
04
응용: 기상청 API 연동 및 데이터 가시화

기상청 단기예보 Open API를 호출하여 실시간 기온 데이터를 수집하고 그래프를 생성합니다.

🤖 AI 프롬프트 가이드 (기상 데이터 처리) "FastAPI 내에서 '기상청_단기예보 조회서비스' API를 호출하는 엔드포인트를 만들고 싶어.
1. requests를 사용해 데이터를 JSON으로 받아줘.
2. 시간별 'T1H(기온)' 항목만 추출해서 Pandas DataFrame으로 변환해줘.
3. x축은 시간(fcstTime), y축은 기온(fcstValue)으로 하는 선 그래프 이미지를 생성해줘.
4. 나눔고딕 등 한글 폰트 설정 코드를 포함해줘."
